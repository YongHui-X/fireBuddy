# FireBuddy Web App

This is the new primary frontend scaffold for FireBuddy.

Use it for:
- Vite-based web-first product work
- Resume-visible deployment targets
- Gradually moving validated logic out of the existing Expo mobile app

Current status:
- This app is scaffolded but intentionally lightweight
- The mobile Expo app in `apps/mobile/` currently has more implemented UI than the web surface
- Shared types and constants should come from `@firebuddy/shared`
